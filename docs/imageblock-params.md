 The image block supports these properties:

  Positional argument:                                                                                                                      
  - expandable — makes the image clickable to open a full-size lightbox overlay
                                                                                                                                            
  Key:value properties:                                                            
  ┌─────────────┬───────────────────────────────────┬─────────────┬─────────────────────────────────────────┐
  │  Property   │              Values               │   Default   │               Description               │
  ├─────────────┼───────────────────────────────────┼─────────────┼─────────────────────────────────────────┤
  │ width       │ pixels (e.g. 400), fill, to-edges │ panel width │ Image width                             │
  ├─────────────┼───────────────────────────────────┼─────────────┼─────────────────────────────────────────┤
  │ float       │ left, right                       │ none        │ Float with text wrapping                │
  ├─────────────┼───────────────────────────────────┼─────────────┼─────────────────────────────────────────┤
  │ ypadding    │ integer (px)                      │ 18          │ Vertical padding (ignored when floated) │
  ├─────────────┼───────────────────────────────────┼─────────────┼─────────────────────────────────────────┤
  │ description │ string                            │ —           │ Alt text for accessibility              │
  ├─────────────┼───────────────────────────────────┼─────────────┼─────────────────────────────────────────┤
  │ caption     │ string                            │ —           │ Italic caption below image              │
  ├─────────────┼───────────────────────────────────┼─────────────┼─────────────────────────────────────────┤
  │ figure      │ string/number                     │ —           │ Figure number label (e.g. "Figure 1")   │
  └─────────────┴───────────────────────────────────┴─────────────┴─────────────────────────────────────────┘
